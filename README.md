# 🎬 SebFlix

**SebFlix** is a modern, feature-rich Movie & TV desktop application designed for a smooth, customizable, and ad-free streaming experience.

---

## 🚀 Features

### 🎥 Content Library
- 50,000+ Movies and TV Shows
- Recently Updated Content
- In-Cinema Listings
- Collections & Curated Categories

### 👤 User Experience
- Multiple Profiles
- Kids Mode
- Watchlists
- Continue Watching
- Saved Playback Progress
- Recently Watched

### 🎨 Customization
- Fully Customizable GUI
- UI Sounds
- Themes & Personalization Options

### 📺 Media & Playback
- In-App Trailers
- Ratings & Detailed Info
- Smooth Playback Experience

### 💻 Platform
- Desktop Application (Electron-based)
- No Ads — Clean Viewing Experience

---

## 📌 Additional Features
- Playlists Creation & Management
- Advanced Content Browsing
- Fast & Responsive Interface
- Constantly Expanding Feature Set
